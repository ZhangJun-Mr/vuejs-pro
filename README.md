# vuejs-pro


## 环境需要
> node 8.9+

## pre start
```
    npm i -g @vue/cli @vue/cli-service-global @vue/cli-service
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
